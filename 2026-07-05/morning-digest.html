Open source + devtools

  • Strix — open-source AI pentesting agent — #1 Python trending today; autonomous agents find, validate, and help fix app vulnerabilities, with CI/CD angle.

https://github.com/usestrix/strix

  • Page Agent by Alibaba — fast-growing in-page GUI agent for controlling web UIs with natural language; useful for SaaS copilots, form filling, and browser/MCP workflows.

https://github.com/alibaba/page-agent

  • Herdr — tmux rebuilt for coding agents — terminal-native multiplexer that shows whether agents are blocked/working/done; good if you run multiple Claude/Codex/Cursor sessions.

https://github.com/ogulcancelik/herdr

AI + coding tools

  • Codex Plugin for Claude Code — lets Claude Code users call Codex for reviews, adversarial review, task rescue, session transfer, and background jobs.

https://github.com/openai/codex-plugin-cc

  • Chrome DevTools MCP v1.5.0 — official Chrome DevTools MCP server for agents; gives coding assistants browser debugging, traces, network, console, screenshots, and automation.

https://github.com/ChromeDevTools/chrome-devtools-mcp

  • TestEvo-Bench — new live benchmark for coding agents on test/code co-evolution: generate or update tests tied to real commits, with executable metrics.

https://arxiv.org/abs/2607.02469

  • Reasoning effort beats tool access for first-try coding reliability — observational study: browser testing tools raised cost without improving score; higher reasoning effort sharply improved perfect first attempts.

https://arxiv.org/abs/2607.02436

  • SkillFuzz — security paper for agent skill marketplaces; fuzzes combinations of “benign” skills to find emergent hidden intents before execution.

https://arxiv.org/abs/2607.02345

AI marketing tools

  • Presenton — open-source AI presentation generator/API with editable PPTX export, custom templates, self-hosting, local models, and MCP support. Practical for sales decks and content ops.

https://github.com/presenton/presenton

  • Short Video Factory — open-source desktop app for batch-generating product marketing short videos with AI copy, TTS, subtitles, and auto-editing.

https://github.com/YILS-LIN/short-video-factory

Learning + memory systems / research

  • LifeOS — Daniel Miessler’s open-source “AI-powered life OS” adds persistent memory, skills, goals, routing, and personal context on top of coding-agent harnesses.

https://github.com/danielmiessler/LifeOS

  • Memora / FAMA benchmark for agent memory — ACL Findings paper tests long-term personalized agents on remembering, reasoning, recommending, and penalizes stale/invalid memories.

https://arxiv.org/abs/2604.20006

Top 3 worth opening first

1. Chrome DevTools MCP — immediately useful for browser-aware coding agents.

2. TestEvo-Bench — strong signal for where coding-agent evals are heading beyond SWE-bench.

3. Presenton — practical open-source marketing/content workflow with API + self-hosting.
