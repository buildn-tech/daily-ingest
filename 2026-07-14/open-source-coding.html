Daily Ingest Dossier — Open Source + Coding Agents

Coding-agent progress today is less about one new autonomous capability than about operating discipline: route models by task, cap unattended spend, and use workflow controls. The sharpest counter-signal is supply-chain risk when agents fetch names they invented.

GPT-5.6 Sol, Terra and Luna in GitHub Copilot

releasemodel routing9 Jul
What it isThree Copilot-selectable variants: Sol for deep work, Terra for daily agentic coding, Luna for cheaper fast jobs.
Why it mattersModel choice can become an explicit operating policy instead of a single default.
Buildn use caseRoute architecture/repo investigation to Sol, normal implementation to Terra, and low-risk chores to Luna.
Useful nowThe rollout is active; Business and Enterprise admins must enable the policy.

Follow-up

Try — Run the same small issue against Terra and Luna; record time, credits and review defects.

Watch-out: Provider-list pricing and gradual rollout mean availability and cost need account-level confirmation.

GitHub Agentic Workflows public preview

previewautomation11 Jun
What it isMarkdown-defined reasoning jobs that compile into GitHub Actions for triage, CI analysis and documentation work.
Why it mattersIt combines reusable agent workflows with default read-only permissions, sandboxing and output checks.
Buildn use casePrototype a weekly stale-issue triage or failed-CI diagnosis workflow in a noncritical repository.
Useful nowThe control model is closer to deployment-ready than an unrestricted autonomous bot.

Follow-up

Open — Read the quickstart and threat-detection documentation before choosing a single no-write pilot.

Watch-out: Public preview; keep merge, credentials and production actions outside the first workflow.

Copilot CLI and SDK session credit limits

releasegovernance1 Jul
What it isPer-session credit caps cover model calls, subagents and background compaction in CLI and SDK runs.
Why it mattersBounded spend is a missing guardrail for unattended coding agents.
Buildn use caseMake a credit cap mandatory on scheduled experiments and delegated maintenance jobs.
Useful nowThe flag can terminate a noninteractive run when a budget is reached.

Follow-up

Save — Add a cap and an explicit stop-state to the next Copilot CLI automation design.

Watch-out: It is a soft cap: in-flight responses can slightly exceed the number set.

Beware of Agentic Botnets: HalluSquatting

papersupply-chain risk8 Jul
What it isA preprint on attackers registering names an agent is likely to hallucinate, then serving hostile content.
Why it mattersThe described path turns an innocent clone or skill install into promptware and possible tool compromise.
Buildn use caseRequire exact canonical URLs, trusted registries and human approval before any agent clones or installs a dependency/skill.
Useful nowTool-enabled agents increasingly fetch Internet resources autonomously; the attack targets that seam.

Follow-up

Turn into task — Add an allowlist and provenance check to Buildn agent bootstrap and skill-install paths.

Watch-out: This is a preprint; treat reported rates as a threat-model signal, not a universal measured constant.
