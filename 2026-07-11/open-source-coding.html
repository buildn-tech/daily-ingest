OpenAI Codex CLI 0.144.x reliability + approval modes

  • Source: openai/codex releases

What it is

A fast-moving terminal coding agent release line; 0.144.1 fixes installer and code-mode reliability, while 0.144.0 adds a write-prompting approval mode and smoother MCP authentication.

Why it matters

Codex is converging on safer local automation: read-only actions can run with less friction while writes still require an explicit boundary.

The July patch fixes the boring but critical parts: installs, macOS code-mode host exposure, and fallbacks when the companion host is unavailable.

A radically simple Python software-engineering agent from the SWE-bench team: bash-only tool use, linear history, subprocess actions, and strong benchmark results.

Why it matters

It is a clean baseline for separating model capability from agent-scaffold complexity.

Useful now
