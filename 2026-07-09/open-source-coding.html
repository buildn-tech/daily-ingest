Daily Ingest Dossier — Open Source + Coding Agents

Coding-agent infrastructure is moving toward smaller harnesses, better control planes, and model/provider routing rather than one monolithic IDE. The strongest signal today is practical: mini-SWE-agent, Crush, Browser Use, OpenHands, and Hermes are all shipping current releases that make agents easier to run, evaluate, route, or embed in existing engineering workflows.
